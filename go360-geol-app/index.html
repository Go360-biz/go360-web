go360-geol-app

Geotechnical workflow application by GO360.biz.

What this app does

go360-geol-app is a geotechnical workflow platform for managing site investigation projects. It lets engineering teams capture site details, review geology and hazard data, track work items, and produce project reports in one system. Google sign-in is used only to authenticate approved users and protect project data. We request basic profile information (name, email, account identifier) to create and manage user sessions, apply access permissions, and attribute actions within the app. go360-geol-app helps geotechnical teams run site assessments and reporting workflows. It uses Google sign-in to securely identify users, enforce access control, and protect project data. Only basic account profile data is used for authentication and user management.
